# Real-Time-News-Impact-Analyzer <br><br>

The Real-Time News Impact Analyzer is an end-to-end AI system that collects live news and social headlines using APIs and web scraping, performs topic detection and sentiment analysis using machine learning, and applies deep learning (CNN) on visual representations of news trends to identify high-impact events. The system provides analytical insights through visual dashboards and supports near real-time updates.
